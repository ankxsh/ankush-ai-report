# AI in 2026: Impact, Adoption & Industry Transformation

An interactive, data-driven presentation analyzing AI's evolution, business impact, India's AI landscape, essential skills for freshers, and industry implementation — built as a single-file HTML application.

## 🚀 Quick Start

1. **Download** or clone this project
2. **Open** `index.html` in any modern browser (Chrome, Edge, Firefox)
3. **Navigate** using `← →` arrow keys or the dot navigation at the bottom

> No installation, no npm, no build step required. Just open and present.

## 🛠️ Tech Stack

| Technology | Purpose |
|-----------|---------|
| **HTML5** | Document structure, semantic markup, canvas elements |

| **Vanilla JavaScript** | Slide navigation, chart rendering, scroll detection |
| **Chart.js v4.4.7** | 8 interactive data visualizations (bar, line, doughnut, radar) |

## ✨ Features

- ⌨️ **Keyboard navigation** — Arrow keys for hands-free presenting
- 📊 **8 interactive charts** — Animated Chart.js visualizations that replay on every visit
- 📱 **Responsive** — Works on desktop, tablet, and projector
- 🖨️ **Print-friendly** — Ctrl+P exports all slides as PDF
- ⬇️ **Scroll indicator** — Auto-detects when slides have scrollable content
- 📝 **Abbreviation boxes** — Full forms included on every slide
- 🇮🇳 **India-specific data** — Dedicated slide on India's AI ecosystem
